Detailed Description

Wraps the functionality of a TCP socket in a class. Errors are returned by throwing exceptions when they occur.

Socket creation is done automatically, so normally you would only need to use Connect(), Bind(), Listen(), Accept(), Close() for setup.

There are a few helper commands for reading/writing data from/to the socket. Namely read_line(), read_header(), read_data().
